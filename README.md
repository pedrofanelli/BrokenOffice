# Backend

## Getting started

1. Clone this repository in your CLI

```
git clone https://github.com/BrokenOffice/Backend-brokenOffice.git
```

2. Install al the project dependencies

```
npm install or npm i
```

3. Seed users and offices into database

```
npm run seed
```

4. Start the server.

```
npm run server
```
